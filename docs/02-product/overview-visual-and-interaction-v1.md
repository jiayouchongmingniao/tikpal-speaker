# Overview 视觉与交互规格 v1

## 定位

`Overview` 是系统主工作台，不是导航页。

目标：

- 一眼看懂 `Listen / Flow / Screen`
- 支持轻操作
- 作为进入全屏模式的入口

## 布局

目标分辨率：`2560 x 720`

第一版建议严格三分屏：

- 左：Listen
- 中：Flow
- 右：Screen

## 三卡共同规则

- 每张卡必须有固定标题：`LISTEN / FLOW / SCREEN`
- 每张卡只保留本模式最本质摘要
- 每张卡都支持：主区进入全屏 + 局部轻操作
- 不允许复用完整页面缩略图

## 视觉原则

- 不是监控大盘
- 不像三个浏览器窗口
- 远距离可识别
- `Flow` 中央卡可承担视觉中心，但第一版不强制非对称布局

## 交互原则

- 单击卡片主区：进入对应全屏
- 单击卡片控件：执行轻操作
- 遥控器左右切高亮，OK 进入
