# 多点触摸手势优先级与误触防护规格 v1

## 第一版推荐手势集

### Overview

- 单击卡片主区：进入全屏
- 单击卡片局部控件：轻操作

### Focus

- 单击空白：显示 / 刷新控制层
- 单击控件：执行操作
- 双指捏合：返回 Overview

## 手势优先级

1. 系统级双指返回
2. 明确命中的局部控件
3. 页面主区单击
4. 空白区单击
5. 可选单指横向滑动

## 误触防护

- 主区与控件区必须明确分离
- 手势都应有位移 / 缩放阈值
- 转场期间锁输入
- overlay 可见时优先命中 overlay 控件

## 模式特殊规则

### Flow

- 单击空白：显示控制层
- 双指捏合：返回 Overview
- 默认不开放复杂手势

### Screen

- 局部控件优先
- 横向切模式更保守

### Listen

- 滑条命中后优先识别为局部拖动
